아이콘 생성 방법

  1. 이 HTML 파일을 브라우저에서 엽니다.
  2. 개발자 도구(F12)를 열고 Elements 탭에서 아이콘 요소를 선택합니다.
  3. 요소를 우클릭하고 "Capture node screenshot"을 선택합니다.
  4. 또는 DevTools에서 Ctrl+Shift+P > "Capture node screenshot"
  5. 저장된 이미지를 해당 크기로 저장합니다:
    • public/icons/icon-192x192.png
    • public/icons/icon-512x512.png
